# AJFX Trading Radar

> Which asset should I be trading today?

## 🚀 Live Dashboard
**Coming soon:** https://ajfx-radar.onrender.com

## Features (Planned)
- ATR-based asset proximity scoring (NQ, ES, YM, GC, CL, DAX)
- Economic news calendar (Forex Factory integration)
- News driver monitoring (Trump tweets, macro catalysts)
- AI-powered market brief per asset
- Twitter/X sentiment on futures-relevant accounts
- HTF level scanner (H1/H4 S/R)
- Trade quality score per asset

## Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** HTML/CSS/JS (React)
- **Hosting:** Render (free tier)
- **CI/CD:** GitHub Actions via Render Blueprint
